# BunCraft Co. — Static Menu Pages

A small static website that presents a restaurant menu for a fictional burger shop (BunCraft Co.). Built with plain HTML and CSS. This project is intended for learning and simple demo use.

## Contents
- `index.html` — Home page / landing page.
- `burgersPage.html` — Burgers menu (active file in workspace).
- `beveragesPage.html` — Beverages menu.
- `dessertsPage.html` — Desserts menu.
- `sidesPage.html` — Sides menu.
- `cartPage.html` — Shopping cart page.
- `nutInfoPage.html` — Allergy / nutrition info page.
- `styles.css` — Shared stylesheet for layout and styling.
- `images/` — Images used across pages.

## How to view
- Quick (no server needed): Open `index.html` in your browser (double-click or `Open File`).
- Or use the VS Code Live Server extension.

## Development notes
- Image filenames contain spaces (e.g., `images/Classic Craft.png`).
